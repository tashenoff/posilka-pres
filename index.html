Архитектура проекта

Клиентское приложение (React Native/Flutter)

Клиенты:

  • Создание заказа
  • Отслеживание заказа
  • Чат с поддержкой

Курьеры:

  • Получение заказов
  • Маршрутизация (API карт)
  • Обновление статуса заказов
  • Чат с поддержкой

Панель оператора (Next.js)

  • Вход для операторов
  • Список заказов
  • Распределение заказов между курьерами
  • Чат с клиентами и курьерами

Бэкенд (Node.js)

API (Express):

  • Управление пользователями
  • Создание/обновление заказов
  • Маршрутизация курьеров

Чат (Socket.io/WebRTC):

  • Реальное время
  • Поддержка общения между ролями

База данных (PostgreSQL/MongoDB)

  • Хранение информации о пользователях
  • Заказы
  • Сообщения (для чата)

Push-уведомления (Firebase Cloud Messaging)

  • Уведомления клиентам о статусе заказов
  • Уведомления курьерам о новых заказах
